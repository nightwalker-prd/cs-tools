# Summary of Phrase Types (أنواع العبارات)

A comprehensive reference for all phrase types in Arabic tarkib.

---

## Overview of Phrase Types

Arabic has several types of phrases, each with specific agreement rules:

```
                        Types of Phrases
                              │
        ┌─────────────────────┼─────────────────────┐
        │                     │                     │
   Agree in DING      Agree Only in I'rab    Do Not Agree
        │                     │                     │
   ┌────┴────┐           ┌────┴────┐          ┌────┴────┐
   │         │           │         │          │         │
Descriptive Demonstrative Conjunctive Appositive Possessive Prepositional
```

## Phrases that Agree in DING

**DING** stands for:
- **D**efinite/Indefinite (مَعْرِفَة/نَكِرَة)
- **I**'rab (grammatical case)
- **N**umber (singular/dual/plural)
- **G**ender (masculine/feminine)

### 1. Descriptive Phrases (التَّرْكِيبُ الوَصْفِي)

| Component | Arabic | Role |
|-----------|--------|------|
| Noun | مَنْعُوتٌ | The word being described |
| Adjective | نَعْتٌ | The describing word |

**Example:**
```
    الوَلَدُ     الصَّالِحُ
   مَنْعُوت      نَعْت
   the pious boy
```

**Rule:** The نَعْت must agree with مَنْعُوت in all four characteristics (DING).

### 2. Demonstrative Phrases (التَّرْكِيبُ الإشَارِي)

| Component | Arabic | Role |
|-----------|--------|------|
| Demonstrative Pronoun | اِسْمُ الإشَارَة | This/that/these/those |
| Pointed-to Noun | مُشَارٌ إِلَيْه | Must have أَلْ |

**Example:**
```
    هٰذَا        الْكِتَابُ
 اسْمُ الإشَارَة   مُشَارٌ إِلَيْه
   this book
```

**Rule:** The مُشَارٌ إِلَيْه must have أَلْ and agree in DING with اِسْمُ الإشَارَة.

## Phrases that Agree Only in I'rab

### 3. Conjunctive Phrases (التَّرْكِيبُ العَطْفِي)

| Component | Arabic | Role |
|-----------|--------|------|
| Conjunction | حَرْفُ عَطْفٍ | وَ، فَ، ثُمَّ، أَوْ |
| Word before conjunction | مَعْطُوفٌ عَلَيْهِ | First item |
| Word after conjunction | مَعْطُوفٌ | Second item |

**Example:**
```
    زَيْدٌ         وَ        خَالِدٌ
 مَعْطُوفٌ عَلَيْهِ   حَرْفُ عَطْفٍ    مَعْطُوفٌ
 Zaid and Khalid
```

**Rule:** The مَعْطُوف agrees in I'rab only with مَعْطُوفٌ عَلَيْهِ.

### 4. Appositive Phrases (التَّرْكِيبُ البَدَلِي)

| Component | Arabic | Role |
|-----------|--------|------|
| First Noun | مُبْدَلٌ مِنْه | The word being explained |
| Second Noun | بَدَل | The explanatory word |

**Example:**
```
    النَّبِيُّ      مُحَمَّدٌ
   مُبْدَلٌ مِنْه     بَدَل
 the Prophet Muhammad
```

**Rule:** The بَدَل agrees in I'rab only with مُبْدَلٌ مِنْه.

---

## Phrases that Do Not Agree

### 5. Possessive Phrases (التَّرْكِيبُ الإضَافِي)

| Component | Arabic | Role | Rule |
|-----------|--------|------|------|
| Owned | مُضَافٌ | First noun | No أَلْ, no تَنْوِين |
| Owner | مُضَافٌ إِلَيْه | Second noun | Always مَجْرُور |

**Example:**
```
    كِتَابُ       الْوَلَدِ
    مُضَافٌ     مُضَافٌ إِلَيْه
  the boy's book
```

**Rules:**
- المُضَاف cannot have أَلْ or تَنْوِين
- المُضَافُ إِلَيْه is always مَجْرُور
- No agreement in DING required

### 6. Prepositional Phrases (شِبْهُ الْجُمْلَةِ)

| Component | Arabic | Role | I'rab |
|-----------|--------|------|-------|
| Particle | جَارٌّ | Preposition | مَبْنِيٌّ |
| Noun | مَجْرُورٌ | Object of preposition | Always مَجْرُور |

**OR:**

| Component | Arabic | Role | I'rab |
|-----------|--------|------|-------|
| Time/Place | مُضَافٌ (ظَرْف) | Adverb | مَنْصُوب, no أَلْ/تَنْوِين |
| Noun | مُضَافٌ إِلَيْه | Genitive | Always مَجْرُور |

**Example:**
```
    فِي        الْبَيْتِ
   جَارٌّ       مَجْرُورٌ
  in the house
```

| Conjunctive | التَّرْكِيبُ العَطْفِي | مَعْطُوفٌ عَلَيْهِ + حَرْفُ عَطْفٍ + مَعْطُوفٌ | I'rab only |

| Prepositional | شِبْهُ الْجُمْلَةِ | جَارٌّ + مَجْرُورٌ | None |

## Detailed Component Reference

### Normal Phrases

| Phrase Type | Component 1 | Component 2 | Agreement Rule |
|-------------|-------------|-------------|----------------|
| Descriptive | مَنْعُوتٌ (Noun) | نَعْتٌ (Adjective) | Agree in DING |
| Demonstrative | اِسْمُ الإشَارَة (Demonstrative Pronoun) | مُشَارٌ إِلَيْه (Must have أَلْ) | Agree in DING |
| Conjunctive | حَرْفُ عَطْفٍ (Conjunction: وَ، فَ، ثُمَّ، أَوْ) | | |
| | مَعْطُوفٌ عَلَيْهِ (Word before conjunction) | مَعْطُوفٌ (Word after conjunction) | Agree in I'rab |
| Appositive | مُبْدَلٌ مِنْه (First Noun) | بَدَل (Second Noun) | Agree in I'rab |

### Possessive Phrases

| Component | Role | Special Rules |
|-----------|------|---------------|
| مُضَافٌ | Owned | No أَلْ, no تَنْوِين |
| مُضَافٌ إِلَيْه | Owner | مَجْرُورٌ (Do not agree) |

### Prepositional Phrases (شِبْهُ الْجُمْلَةِ)

| Component | Role | I'rab |
|-----------|------|-------|
| مُضَافٌ (Time/Place: ظَرْف) | Adverb | مَنْصُوب, no أَلْ, no تَنْوِين |
| مُضَافٌ إِلَيْه | Noun | مَجْرُورٌ |
| جَارٌّ | Particle | مَبْنِيٌّ |
| مَجْرُورٌ | Noun | مَجْرُورٌ |
