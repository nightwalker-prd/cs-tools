## Pronouns as مُؤَكَّد

### Types of Pronouns in Emphasis

The مُؤَكَّد (emphasized word) can be a ضَمِير (pronoun), which comes in two types:

1. **ضَمِير مُنْفَصِل** (separate/independent pronoun)
2. **ضَمِير مُتَّصِل** (attached pronoun)

---

### ضَمِير مُنْفَصِل as مُؤَكَّد

When the مُؤَكَّد is a **separate pronoun**, its تَأْكِيد لَفْظِيّ must also be a separate pronoun with the same i'rab.

#### Examples

| Arabic | Translation | Analysis |
|--------|-------------|----------|
| إِيَّاهُ إِيَّاهُ أَعْبُدُ | It is **He whom** I worship | إِيَّاهُ repeated |
| أَنْتَ أَنْتَ قَائِمٌ | You are standing (emphatic) | أَنْتَ repeated |

#### Tarkib Diagram

```
┌─────────────┬─────────────┬─────────────┐
│    مُؤَكَّد    │    تَأْكِيد    │   Remainder │
├─────────────┼─────────────┼─────────────┤
│   إِيَّاهُ    │   إِيَّاهُ    │   أَعْبُدُ    │
└─────────────┴─────────────┴─────────────┘
```

---

### ضَمِير مُتَّصِل as مُؤَكَّد

When the مُؤَكَّد is an **attached pronoun**, the تَأْكِيد will **NOT** be a ضَمِير مُنْفَصِل. Instead, use the appropriate attached form.

#### Critical Rule

| Correct ✓ | Incorrect ✗ | Explanation |
|-----------|-------------|-------------|
| ذَهَبْتُ أَنَا | ذَهَبْتُ | Need separate pronoun to emphasize |
| رَأَيْتُكَ أَنْتَ | رَأَيْتُكَكَ | Cannot double attached pronouns |

#### Pattern for Different Pronoun Types

| Attached مُؤَكَّد | تَأْكِيد | Example | Translation |
|------------------|---------|---------|-------------|
| ـتُ (I) | أَنَا | ذَهَبْتُ أَنَا | I myself went |
| ـكَ (you m.) | أَنْتَ | رَأَيْتُكَ أَنْتَ | I saw you yourself |
| ـكِ (you f.) | أَنْتِ | سَمِعْتُكِ أَنْتِ | I heard you yourself |
| ـهُ (him) | هُوَ | رَأَيْتُهُ هُوَ | I saw him himself |
| ـهَا (her) | هِيَ | رَأَيْتُهَا هِيَ | I saw her herself |
| ـنَا (us) | نَحْنُ | رَأَيْتَنَا نَحْنُ | You saw us ourselves |
| ـهُمْ (them) | هُمْ | رَأَيْتُهُمْ هُمْ | I saw them themselves |

#### Tarkib Diagrams

**For فَاعِل emphasis:**
```
┌─────────────────┬─────────────┬─────────────┐
│  فِعْل وَفَاعِل (ت)  │    مُؤَكَّد    │    تَأْكِيد    │
├─────────────────┼─────────────┼─────────────┤
│     ذَهَبْـ      │      تُ      │     أَنَا     │
└─────────────────┴─────────────┴─────────────┘
```

**For مَفْعُول بِهِ emphasis:**
```
┌─────────────────┬─────────────┬─────────────┐
│  فِعْل وَفَاعِل (ت)  │    مُؤَكَّد    │    تَأْكِيد    │
├─────────────────┼─────────────┼─────────────┤
│     رَأَيْـ      │      كَ      │    أَنْتَ    │
└─────────────────┴─────────────┴─────────────┘
```

---

### ضَمِير مُسْتَتِر (Hidden Pronoun) as مُؤَكَّد

If the فَاعِل pronoun is **hidden** (ضَمِير مُسْتَتِر), it must be written after the verb in brackets when showing the تَأْكِيد.

#### Example

| Arabic | Explanation |
|--------|-------------|
| ذَهَبَ هُوَ | He went (emphatic) |

#### Tarkib Diagram

```
┌─────────────┬─────────────┬─────────────┐
│    فِعْل     │    فَاعِل    │             │
├─────────────┼─────────────┼─────────────┤
│    ذَهَبَ    │    (هُوَ)    │             │
│             │     هُوَ     │             │
│             │    مُؤَكَّد    │    تَأْكِيد    │
└─────────────┴─────────────┴─────────────┘
```

## Coordinated Phrases (العَطْف)

### Rule: مَعْطُوف عَلَيْهِ as ضَمِير الفَاعِل

When the **مَعْطُوف عَلَيْهِ** (the word being coordinated from) is the **ضَمِير الفَاعِل** (subject pronoun), you must:

1. Repeat the corresponding مُبْتَدَأ pronoun (ضَمِير مَرْفُوع مُنْفَصِل)
2. Place it **before** the حَرْف العَطْف
3. Form a مُؤَكَّد-تَأْكِيد structure
4. Then add the مَعْطُوف

#### Example

| Correct ✓ | Incorrect ✗ |
|-----------|-------------|
| ذَهَبْتُ أَنَا وَزَيْدٌ | ذَهَبْتُ وَزَيْدٌ |

**Translation:** *Zaid and I went.* (NOT *I and Zaid went.* in English)

#### Why This Rule Exists

In Arabic, the attached pronoun (ـتُ) cannot be directly coordinated with a noun. The separate pronoun (أَنَا) must be brought to serve as the proper مَعْطُوف عَلَيْهِ.

#### Tarkib Diagram

```
┌─────────┬───────────────────────────────────────────┐
│  فِعْل   │                  فَاعِل                    │
├─────────┼─────────┬─────────┬─────────┬─────────────┤
│ ذَهَبْـ  │    تُ    │   أَنَا   │    وَ    │    زَيْدٌ    │
│         │  مُؤَكَّد  │  تَأْكِيد  │ حَرْف عَطْف │   مَعْطُوف   │
│         │         │  مَعْطُوف عَلَيْهِ  │         │             │
└─────────┴─────────┴─────────┴─────────┴─────────────┘
```

#### More Complex Example

```
لَ تَسْمَعَ جَهِي وَ بِنْتُهَا
(You will not hear me and her daughter)
```

```
┌───────┬─────────┬─────────────┬─────────┬──────────┬─────────┬──────────┐
│لَام الابتداء│  فِعْل   │  فَاعِل ...  │ تَوْن التَّأْكِيد│ مَفْعُول بِهِ│ حَرْف عَطْف│ مَعْطُوف عَلَيْهِ│
├───────┼─────────┼─────────────┼─────────┼──────────┼─────────┼──────────┤
│   لَ   │  تَسْمَعَ │    (هِيَ)    │    نَّ    │   جَهِي   │    وَ    │  بِنْتُهَا  │
│       │         │             │         │   مُؤَكَّد  │         │          │
│       │         │             │  تَأْكِيد  │          │         │          │
│       │         │             │  مُؤَكَّد  │          │         │          │
└───────┴─────────┴─────────────┴─────────┴──────────┴─────────┴──────────┘
```

---

### Exception: Word Between مَعْطُوف عَلَيْهِ and حَرْف عَطْف

If there is **another word** between the مَعْطُوف عَلَيْهِ and حَرْف عَطْف, there is **no need** to add a ضَمِير مَرْفُوع مُنْفَصِل.

#### Example

| Arabic | Translation |
|--------|-------------|
| مَا ذَهَبْتُ وَلَا زَيْدٌ | Neither Zaid nor I went |

**Explanation:** The particle لَا comes between the مَعْطُوف عَلَيْهِ (تُ) and مَعْطُوف (زَيْدٌ), so there is no need to bring the pronoun أَنَا.

## Verbal Emphasis Rules

### تَأْكِيد لَفْظِيّ with Pronouns

#### Rule 1: Matching I'rab

When مُؤَكَّد is ضَمِير مَجْرُور (genitive pronoun) with a حَرْف جَارّ, the تَأْكِيد will be a ضَمِير مَرْفُوع مُنْفَصِل, or the entire شِبْهُ الجُمْلَة (prepositional phrase) will be repeated.

| Arabic | Translation |
|--------|-------------|
| أَنْظُرُ إِلَيْكَ أَنْتَ | It is you who I am looking at |
| أَنْظُرُ إِلَيْكَ إِلَيْكَ | It is you who I am looking at |

#### Rule 2: Mixed Pronoun Types

The تَأْكِيد لَفْظِيّ of a تَأْكِيد لَفْظِيّ can combine:
- ضَمِير مَنْصُوب مُتَّصِل (attached accusative pronoun)
- ضَمِير مَجْرُور مُتَّصِل (attached genitive pronoun)
- ضَمِير مَرْفُوع مُنْفَصِل (separate nominative pronoun)

| Arabic | Translation |
|--------|-------------|
| إِنَّهُ هُوَ اللهُ | Indeed, it is He who is Allah |
| هَذَا كِتَابِي أَنَا | This is my book (emphatic) |

## Practice Patterns

### Pattern 1: Simple فَاعِل Emphasis

```
فِعْل + فَاعِل (attached) + تَأْكِيد (separate)
ذَهَبْتُ + أَنَا
```

### Pattern 2: مَفْعُول بِهِ Emphasis

```
فِعْل + فَاعِل + مَفْعُول بِهِ (attached) + تَأْكِيد (separate)
رَأَيْتُ + كَ + أَنْتَ
```

### Pattern 3: Coordination

```
فِعْل + فَاعِل (attached) + تَأْكِيد (separate) + وَ + مَعْطُوف
ذَهَبْتُ + أَنَا + وَ + زَيْدٌ
```

### Pattern 4: Hidden Pronoun

```
فِعْل + (فَاعِل hidden) + تَأْكِيد (separate)
ذَهَبَ + (هُوَ) + هُوَ
```

---

## Key Takeaways

1. **Attached pronouns** need a **separate pronoun** as their تَأْكِيد
2. **Separate pronouns** repeat themselves as تَأْكِيد
3. **Hidden pronouns** are written in brackets and emphasized with separate pronouns
4. **Coordination** requires bringing the separate pronoun before the حَرْف عَطْف
5. The **exception** to coordination rule is when another word separates the elements
