## Question 146

**Arabic Text:**
﴿وَعَدَ اللهُ الْمُؤْمِنِينَ وَالْمُؤْمِنَاتِ جَنَّاتٍ تَجْرِي مِنْ تَحْتِهَا الْأَنْهَارُ ... وَمَسَاكِنَ طَيِّبَةً فِي جَنَّاتِ عَدْنٍ﴾

**Translation:**
Allah has promised the believing men and believing women gardens beneath which rivers flow... and pleasant dwellings in the Gardens of Eden.

**Tarkib Analysis:**

| Word | Analysis |
|------|----------|
| وَعَدَ | فِعْل مَاضٍ |
| اللهُ | فَاعِل (مَرْفُوع) |
| الْمُؤْمِنِينَ | مَفْعُول بِهِ أَوَّل (مَنْصُوب) |
| وَ | حَرْف عَطْف |
| الْمُؤْمِنَاتِ | مَعْطُوف (مَنْصُوب) |
| جَنَّاتٍ | مَفْعُول بِهِ ثَانٍ (مَنْصُوب) |
| تَجْرِي | فِعْل مُضَارِع |
| مِنْ | حَرْف جَرّ |
| تَحْتِ | مَجْرُور (مُضَاف) |
| ـهَا | مُضَاف إِلَيْهِ |
| الْأَنْهَارُ | فَاعِل (مَرْفُوع) |
| **تَجْرِي مِنْ تَحْتِهَا الْأَنْهَارُ** | **جُمْلَة فِعْلِيَّة = نَعْت لِـ جَنَّاتٍ** |
| وَ | حَرْف عَطْف |
| مَسَاكِنَ | مَعْطُوف عَلَى جَنَّاتٍ (مَنْصُوب) |
| طَيِّبَةً | نَعْت (مَنْصُوب) |
| فِي | حَرْف جَرّ |
| جَنَّاتِ | مَجْرُور (مُضَاف) |
| عَدْنٍ | مُضَاف إِلَيْهِ (مَجْرُور) |

## Question 147

**Arabic Text:**
ذَالِكَ الْكِتَابُ ﴿هُدًى لِلْمُتَّقِينَ ۞ الَّذِينَ يُؤْمِنُونَ بِالْغَيْبِ وَيُقِيمُونَ الصَّلَاةَ وَمِمَّا رَزَقْنَاهُمْ يُنْفِقُونَ ۞﴾

**Translation:**
That is the Book - guidance for the God-conscious. Those who believe in the unseen, establish prayer, and spend from what We have provided them.

**Tarkib Analysis:**

| Word | Analysis |
|------|----------|
| ذَالِكَ | اسْم إِشَارَة (مُبْتَدَأ) |
| الْكِتَابُ | مُشَار إِلَيْهِ / بَدَل (مَرْفُوع) |
| هُدًى | خَبَر (مَرْفُوع - مُقَدَّر) |
| لِ | حَرْف جَرّ |
| الْمُتَّقِينَ | مَجْرُور |
| الَّذِينَ | اسْم مَوْصُول (نَعْت لِلْمُتَّقِينَ أَوْ بَدَل) |
| يُؤْمِنُونَ | فِعْل مُضَارِع + فَاعِل (وَاو الجَمَاعَة) |
| بِـ | حَرْف جَرّ |
| الْغَيْبِ | مَجْرُور |
| **يُؤْمِنُونَ بِالْغَيْبِ** | **صِلَة المَوْصُول** |
| وَ | حَرْف عَطْف |
| يُقِيمُونَ | فِعْل مُضَارِع + فَاعِل (مَعْطُوف) |
| الصَّلَاةَ | مَفْعُول بِهِ (مَنْصُوب) |
| وَ | حَرْف عَطْف |
| مِمَّا | مِنْ + مَا (حَرْف جَرّ + اسْم مَوْصُول) |
| رَزَقْنَا | فِعْل مَاضٍ + فَاعِل (نَا) |
| ـهُمْ | مَفْعُول بِهِ (ضَمِير مُتَّصِل) |
| يُنْفِقُونَ | فِعْل مُضَارِع + فَاعِل |

## Question 149

**Arabic Text:**
﴿إِنَّ الَّذِينَ قَالُوا رَبُّنَا اللهُ ثُمَّ اسْتَقَامُوا تَتَنَزَّلُ عَلَيْهِمُ الْمَلَائِكَةُ﴾ عِنْدَ الْمَوْتِ يَقُولُونَ: لَا ﴿تَخَافُوا وَلَا تَحْزَنُوا وَأَبْشِرُوا بِالْجَنَّةِ الَّتِي كُنْتُمْ تُوعَدُونَ﴾

**Translation:**
Indeed, those who say "Our Lord is Allah" and then remain steadfast - upon them the angels descend. At the time of death they say: Do not fear and do not grieve, and receive good tidings of Paradise which you were promised.

**Tarkib Analysis:**

| Word | Analysis |
|------|----------|
| إِنَّ | حَرْف مُشَبَّه بِالْفِعْل |
| الَّذِينَ | اسْم مَوْصُول (اسْم إِنَّ - مَنْصُوب مَحَلًّا) |
| قَالُوا | فِعْل مَاضٍ + فَاعِل (صِلَة المَوْصُول) |
| رَبُّ | مُبْتَدَأ (مُضَاف) |
| ـنَا | مُضَاف إِلَيْهِ |
| اللهُ | خَبَر |
| **رَبُّنَا اللهُ** | **مَقُول القَوْل (مَفْعُول بِهِ)** |
| ثُمَّ | حَرْف عَطْف |
| اسْتَقَامُوا | فِعْل مَاضٍ + فَاعِل (مَعْطُوف عَلَى قَالُوا) |
| تَتَنَزَّلُ | فِعْل مُضَارِع |
| عَلَى | حَرْف جَرّ |
| ـهِمُ | مَجْرُور (ضَمِير مُتَّصِل) |
| الْمَلَائِكَةُ | فَاعِل (مَرْفُوع) |
| **تَتَنَزَّلُ عَلَيْهِمُ الْمَلَائِكَةُ** | **خَبَر إِنَّ** |
| لَا | حَرْف نَهْي |
| تَخَافُوا | فِعْل مُضَارِع مَجْزُوم + فَاعِل |
| وَلَا | حَرْف عَطْف + حَرْف نَهْي |
| تَحْزَنُوا | فِعْل مُضَارِع مَجْزُوم + فَاعِل |
| وَ | حَرْف عَطْف |
| أَبْشِرُوا | فِعْل أَمْر + فَاعِل |
| بِـ | حَرْف جَرّ |
| الْجَنَّةِ | مَجْرُور |
| الَّتِي | اسْم مَوْصُول (نَعْت) |
| كُنْتُمْ | فِعْل نَاقِص + اسْم كَانَ |
| تُوعَدُونَ | فِعْل مُضَارِع مَبْنِيّ لِلْمَجْهُول (خَبَر كَانَ) = صِلَة المَوْصُول |

## Question 150 (Part 1: خَبَر)

**Arabic Text:**
﴿أُولَئِكَ جَزَاؤُهُمْ أَنَّ عَلَيْهِمْ لَعْنَةَ اللهِ وَالْمَلَائِكَةِ وَالنَّاسِ أَجْمَعِينَ﴾

**Translation:**
Those - their recompense is that upon them is the curse of Allah and the angels and all mankind.

**Tarkib Analysis:**

| Word | Analysis |
|------|----------|
| أُولَئِكَ | اسْم إِشَارَة (مُبْتَدَأ أَوَّل) |
| جَزَاؤُ | مُبْتَدَأ ثَانٍ (مُضَاف) |
| ـهُمْ | مُضَاف إِلَيْهِ |
| أَنَّ | حَرْف مُشَبَّه بِالْفِعْل |
| عَلَى | حَرْف جَرّ |
| ـهِمْ | مَجْرُور |
| **عَلَيْهِمْ** | **خَبَر أَنَّ مُقَدَّم (شِبْه جُمْلَة)** |
| لَعْنَةَ | اسْم أَنَّ (مُضَاف) |
| اللهِ | مُضَاف إِلَيْهِ |
| وَ | حَرْف عَطْف |
| الْمَلَائِكَةِ | مَعْطُوف (مَجْرُور) |
| وَ | حَرْف عَطْف |
| النَّاسِ | مَعْطُوف (مَجْرُور) |
| أَجْمَعِينَ | تَوْكِيد (تَأْكِيد مَعْنَوِيّ) |
| **أَنَّ عَلَيْهِمْ لَعْنَةَ اللهِ...** | **خَبَر المُبْتَدَأ الثَّانِي (جَزَاؤُهُمْ)** |
| **جَزَاؤُهُمْ أَنَّ عَلَيْهِمْ...** | **جُمْلَة اسْمِيَّة = خَبَر المُبْتَدَأ الأَوَّل (أُولَئِكَ)** |

| Word | Analysis |
|------|----------|

| ـنَا | مَفْعُول بِهِ (ضَمِير مُتَّصِل) |
| رَبُّ | فَاعِل (مُضَاف) |
| ـنَا | مُضَاف إِلَيْهِ |
| (الْجَنَّةَ) | مَفْعُول بِهِ ثَانٍ - مَحْذُوف |
| مَعَ | ظَرْف (مُضَاف) |
| الْقَوْمِ | مُضَاف إِلَيْهِ |

## Summary of Key Tarkib Concepts Used

### Sentence Types:
- **جُمْلَة اسْمِيَّة** (Nominal Sentence): مُبْتَدَأ + خَبَر
- **جُمْلَة فِعْلِيَّة** (Verbal Sentence): فِعْل + فَاعِل + (مَفْعُول بِهِ)

### Particles:
- **حَرْف مُشَبَّه بِالْفِعْل**: إِنَّ، أَنَّ، لَكِنَّ (causes اسْم to be مَنْصُوب, خَبَر مَرْفُوع)
- **حَرْف شَرْط جَازِم**: إِنْ، مَنْ (causes جَزْم of both فِعْل الشَّرْط and جَوَاب الشَّرْط)
- **حَرْف مَصْدَرِيّ نَاصِب**: أَنْ (creates مَصْدَر مُؤَوَّل)
- **فَاء السَّبَبِيَّة**: causes the following verb to be مَنْصُوب

### Phrase Types:
- **اسْم مَوْصُول + صِلَة**: الَّذِي، الَّتِي، الَّذِينَ، مَنْ، مَا
- **مُضَاف + مُضَاف إِلَيْهِ**: Possessive construction
- **مَنْعُوت + نَعْت**: Descriptive phrase

### Special Constructions:
- **جَوَاب الأَمْر**: Response to command (مَجْزُوم)
- **جَوَاب الشَّرْط**: Response to condition (مَجْزُوم)
- **مَفْعُول لَهُ**: Purpose/reason for action
- **جُمْلَة صُغْرَى**: Subordinate sentence functioning as a grammatical unit
