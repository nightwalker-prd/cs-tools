## Particles (حُرُوفُ جَارَّة)

Prepositions that make the following noun مَجْرُور.

| Arabic | English | Common Uses |
|--------|---------|-------------|
| فِي | in | Location, time |
| عَلَى | on | Surface, obligation |
| إِلَى | to, towards, until | Direction, limit |
| مِنْ | from | Origin, partitive |
| بِ | with | Instrument, accompaniment |
| لِ | for, belongs to | Purpose, possession |
| حَتَّى | until, even | Limit |
| عَنْ | regarding, from | Separation, about |
| كَ | like | Comparison |

## Key Grammatical Terms

| Arabic | English | Definition |
|--------|---------|------------|
| حَرْفُ صِلَةٍ | Particle of emphasis | بِ or مِنْ used for emphasis |
| إِعْرَابٌ لَفْظِيٌّ | Apparent I'rab | Visible case ending |
| إِعْرَابٌ مَحَلِّيٌّ | Inferred I'rab | Assumed case based on function |
| فِي مَحَلِّ الْجَرِّ | In the slot of genitive | Inferred genitive case |
| فِي مَحَلِّ النَّصْبِ | In the slot of accusative | Inferred accusative case |
| فِي مَحَلِّ الرَّفْعِ | In the slot of nominative | Inferred nominative case |
| مَبْنِيٌّ | Indeclinable | Word that doesn't change ending |
| لَا لِنَفْيِ الْجِنْسِ | لا for negating genus | Creates emphatic negative meaning |
| شِبْهُ الْجُمْلَةِ | Prepositional Phrase | جَارٌّ + مَجْرُورٌ |
| غَيْرُ صَرِيحٍ | Indirect slot | Non-explicit grammatical function |
| ظَرْفٌ | Noun Preposition | Time/place adverb |
| خَبَرٌ مُقَدَّمٌ | Fronted predicate | Predicate that comes before subject |
| مُبْتَدَأٌ مُؤَخَّرٌ | Delayed subject | Subject that comes after predicate |
