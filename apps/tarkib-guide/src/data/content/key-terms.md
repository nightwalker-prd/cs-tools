# Key Terms and Vocabulary (المُصْطَلَحَاتُ وَالمُفْرَدَاتُ)

## Grammatical Key Terms

| English | Arabic | Description |
|---------|--------|-------------|
| Imperative | الْأَمْرُ | Command form |
| Third-person imperative | الْأَمْرُ لِلْغَائِبِ | Indirect command (Let him...) |
| Negative imperative (Prohibition) | النَّهْيُ | Prohibition (Don't...) |
| Particle for negative imperative (لا) | حَرْفُ نَهْيٍ | The لا of prohibition |
| Pronoun | ضَمِيرٌ ج ضَمَائِر | A word replacing a noun |
| First person | مُتَكَلِّم | The speaker (I, we) |
| Second person | مُخَاطَب | The addressee (you) |
| Third person | غَائِب | The absent one (he, she, they) |
| Hidden pronoun | ضَمِيرٌ مُسْتَتِر | Implied in the verb |
| Visible pronoun | ضَمِيرٌ بَارِز | Explicitly visible |
| Attached pronoun | ضَمِيرٌ مُتَّصِل | Connected to another word |
| Detached pronoun | ضَمِيرٌ مُنْفَصِل | Stands alone |
| Pronoun of Separation | ضَمِيرُ الْفَصْلِ | Differentiates خَبَر from نَعْت |
| Reflexive pronouns | ضَمِيرٌ مَرْفُوعٌ + نَفْس | Pronouns referring to oneself |
| Reciprocal pronouns | بَعْض + بَعْضًا | Pronouns meaning "each other" |
| Nominative Pronouns | ضَمِيرٌ مَرْفُوعٌ | Subject pronouns |
| Accusative Pronouns | ضَمِيرٌ مَنْصُوبٌ | Object pronouns |
| Genitive Pronouns | ضَمِيرٌ مَجْرُورٌ | Pronouns after prepositions |
| A نُون to express the فَاعِل state | نُونُ الْإِعْرَابِ | Nun indicating nominative case |
| A نُون for the feminine pronoun | نُونُ النِّسْوَةِ / نُونُ الضَّمِيرِ | Nun indicating feminine |
| Particle of illness (حَرْف عِلَّة) | يَاءٌ or وَاوٌ ,أَلِفٌ | Weak letters |

## Common Patterns in Tarkib

### Sentence Components

| Component | Arabic | Function |
|-----------|--------|----------|
| Subject (nominal) | مُبْتَدَأ | What the sentence is about |
| Predicate | خَبَر | Information about the subject |
| Verb | فِعْل | The action |
| Doer/Subject | فَاعِل | Who performs the action |
| Object | مَفْعُولٌ بِهِ | What receives the action |
| Preposition | حَرْفُ جَرٍّ / جَارّ | Governing particle |
| Genitive noun | مَجْرُور | Noun after preposition |
| Possessor | مُضَافٌ إِلَيْهِ | Second part of possession |
| Possessed | مُضَاف | First part of possession |

## Pronoun Quick Reference

### Nominative (مَرْفُوع)

| Person | Detached | Attached (Past) |
|--------|----------|-----------------|
| He | هُوَ | فَعَلَ (hidden) |
| She | هِيَ | فَعَلَتْ (hidden) |
| They (m) | هُمْ | فَعَلُوا |
| You (m.s.) | أَنْتَ | فَعَلْتَ |
| I | أَنَا | فَعَلْتُ |
| We | نَحْنُ | فَعَلْنَا |

### Accusative (مَنْصُوب)

| Person | Attached | Detached |
|--------|----------|----------|
| Him | ـهُ | إِيَّاهُ |
| Her | ـهَا | إِيَّاهَا |
| Them (m) | ـهُمْ | إِيَّاهُمْ |
| You (m.s.) | ـكَ | إِيَّاكَ |
| Me | ـنِي | إِيَّايَ |
| Us | ـنَا | إِيَّانَا |

### Genitive (مَجْرُور)

| Person | After Preposition | As مُضَافٌ إِلَيْهِ |
|--------|-------------------|---------------------|
| Him | لَهُ | كِتَابُهُ |
| Her | لَهَا | كِتَابُهَا |
| Them (m) | لَهُمْ | كِتَابُهُمْ |
| You (m.s.) | لَكَ | كِتَابُكَ |
| Me | لِي | كِتَابِي |
| Us | لَنَا | كِتَابُنَا |

## Summary of Special Pronoun Types

| Type | Formation | Translation | Example |
|------|-----------|-------------|---------|
| **Reflexive** | نَفْس + pronoun | -self | نَفْسَهُ = himself |
| **Reciprocal** | بَعْض... بَعْضًا | each other | يَنْصُرُ بَعْضُهُمْ بَعْضًا |
| **Separation** | Pronoun between مُبْتَدَأ and خَبَر | (emphasis) | هُمُ الصَّالِحُونَ |
