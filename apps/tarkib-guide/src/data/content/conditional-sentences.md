# Conditional Sentences (الشَّرْط)

## Overview

A conditional sentence is comprised of three parts:

| Component | Arabic Term | Description |
|-----------|-------------|-------------|
| **أَدَاةُ الشَّرْطِ** | Conditional conjunction | Words like if, when, whenever, whoever |
| **شَرْط** | Condition clause | The "if" part |
| **جَوَابُ الشَّرْطِ** | Result clause | The "then" part |

**Example:**
```
إِنْ تَجْتَهِدْ تَنْجَحْ
If you work hard, you will be successful.
```

- **إِنْ** = أَدَاةُ الشَّرْطِ (conditional conjunction)
- **تَجْتَهِدْ** (you work hard) = شَرْط (condition)
- **تَنْجَحْ** (you will be successful) = جَوَابُ الشَّرْطِ (result)

---

## Conditional Conjunctions (أَدَاةُ الشَّرْطِ)

There are two types of conditional conjunctions:

### 1. Noun Conditional Conjunctions (أَسْمَاءُ الشَّرْطِ)

There are nine أَسْمَاءُ الشَّرْطِ (also used as أَسْمَاءُ الاسْتِفْهَام):

| Arabic | English | Arabic | English |
|--------|---------|--------|---------|
| مَنْ | whoever | كَيْفَ | however |
| مَا | whatever | أَنَّى | from wherever, however |
| أَيْنَ | wherever | كَمْ | however many |
| مَتَى | whenever | أَيّ | whichever |
| أَيَّانَ | whenever | | |

> **Note:** The أَسْمَاءُ الشَّرْطِ are عَامِل—they render the مُضَارِع in both the condition and result to be in the مَجْزُوم state.

### 2. Particle Conditional Conjunctions (حُرُوفُ الشَّرْطِ)

There are three حُرُوفُ الشَّرْطِ:

| Arabic | English | Notes |
|--------|---------|-------|
| إِنْ | if | عَامِل - renders مُضَارِع in both clauses مَجْزُوم |
| لَوْ | if | غَيْرُ عَامِل (for unreal conditionals) |
| لَوْلَا | if not, if it were not for | غَيْرُ عَامِل |

---

## Types of Conditional Sentences

1. **Real conditionals**: Conditions which may be met
2. **Unreal conditionals**: Conditions which cannot be met

---

## Real Conditionals

Real conditionals are constructed using either of the following أَدَاةُ الشَّرْطِ:

### 1. Using أَسْمَاءُ الشَّرْطِ

```
مَنْ يَجْتَهِدْ يَنْجَحْ
Whoever works hard will be successful.
```

The tarkib of أَسْمَاءُ الشَّرْطِ is the same as أَسْمَاءُ الاسْتِفْهَام:

| شَرْط | | جَوَابُ الشَّرْطِ |
|-------|--|------------------|
| مُبْتَدَأ | خَبَر | فِعْل وَفَاعِل (هُوَ) |
| مَنْ | يَجْتَهِدُ | يَنْجَحْ |

### 2. Using إِنْ (Particle)

```
إِنْ تَجْتَهِدْ تَنْجَحْ
If you work hard, you will be successful.
```

In tarkib, إِنْ is labelled as حَرْفُ شَرْطٍ:

| | شَرْط | جَوَابُ الشَّرْطِ |
|--|-------|------------------|
| حَرْفُ شَرْطٍ | فِعْل وَفَاعِل (أَنْتَ) | فِعْل وَفَاعِل (أَنْتَ) |
| إِنْ | تَجْتَهِدْ | تَنْجَحْ |

---

## Translation of Real Conditionals

There are two types of real conditionals:

### Zero Conditionals (General Truths)
General truths and situations that occur all the time. The present tense is used in both clauses:
```
If you work hard, you are successful.
```

### First Conditionals (Future Possible)
Future situations which may occur. Present tense in the condition clause, future tense in the result clause:
```
If you work hard, you will be successful.
```

**Note:** If the condition clause is mentioned first, a comma is placed between the two clauses. If the result clause is mentioned first, the comma is omitted.

---

## Additional Rules of Condition and Result Clauses

### The Verb of the Condition Clause (شَرْط)

The verb in the شَرْط is primarily a فِعْل مُضَارِع which is مَجْزُوم:
```
إِنْ تَصْدُقْ
If you speak the truth
```

However, it can also be:

#### 1. A فِعْل مَاضٍ
Translated as a فِعْل مُضَارِع:
```
إِنْ صَدَقْتَ
If you speak the truth
```

#### 2. The كَانَ of فِعْل نَاقِص
Not translated as was/were:
```
إِنْ كُنْتَ صَادِقًا
If you are truthful
```

**Summary of شَرْط Forms:**

| مُضَارِع | مَاضٍ | كَانَ |
|---------|-------|-------|
| مَنْ يَفْعَلَ... | مَنْ فَعَلَ... | مَنْ كَانَ فَاعِلًا... |

---

### The Verb of the Result Clause (جَوَابُ الشَّرْطِ)

The جَوَابُ الشَّرْطِ is primarily:

#### 1. A فِعْل مُضَارِع in the مَجْزُوم state
Without any additions like لَنْ، سَـ, etc.:
```
إِنْ تَجْتَهِدْ تَنْجَحْ
If you work hard, you are successful.
```

#### 2. A فِعْل مَاضٍ with a مُضَارِع meaning:
```
إِنِ اجْتَهَدْتَ نَجَحْتَ
If you work hard, you are successful.
```

### When جَوَابُ الشَّرْطِ Requires فَ

If the جَوَابُ الشَّرْطِ is neither of the above two, it will be preceded by **فَ**:
```
إِنْ يَضْرِبْكَ زَيْدٌ فَلَا تَضْرِبْهُ
If Zaid hits you, do not hit him.
```

This فَ is not translated, and in tarkib it is labelled as **رَابِطَة** (connector)—used to join the شَرْط with the جَوَاب.

| شَرْط | | رَابِطَة | جَوَابُ شَرْطٍ |
|-------|--|---------|---------------|
| فِعْل وَمَفْعُولٌ بِهِ | فَاعِل | | فِعْل وَفَاعِل (أَنْتَ) مَفْعُولٌ بِهِ |
| يَضْرِبْكَ | زَيْدٌ | فَ | لَا تَضْرِبْهُ |

---

## Summary: Forms of جَوَابُ شَرْطٍ

| Type | Form | Example |
|------|------|---------|
| مُضَارِع مَجْزُوم | Without additions | يَنْجَحْ |
| مَاضٍ | With مُضَارِع meaning | نَجَحَ |
| Other | Preceded by فَ | فَهُوَ نَاجِحٌ، فَسَيَنْجَحُ، ... |

---

## Negating the Verb in the Condition Clause

The verb of the شَرْط is made negative by adding the particle **لَمْ**.
The verb of the جَوَابُ الشَّرْطِ is made negative by adding **لَا** or **مَا**.

```
إِنْ لَمْ تَجْتَهِدْ لَا تَنْجَحْ
If you do not work hard, you will not be successful.
```

The negative verb in a conditional can be translated as "unless":
```
Unless you work hard, you will not be successful.
```

---

## جَوَابٌ مُقَدَّم (Fronted Result Clause)

Sometimes, in a conditional sentence, the جَوَاب may precede the شَرْط. In this case, the أَدَاةُ الشَّرْطِ will not cause the جَوَاب to be مَجْزُوم.

```
أَجْتَهِدُ إِنِ اجْتَهَدْتَ
I will work hard if you work hard.
```

The tarkib is written as follows:

| جَوَابُ شَرْطٍ مُقَدَّم | | شَرْطٌ مُؤَخَّر |
|----------------------|--|----------------|
| فِعْل وَفَاعِل (أَنَا) | حَرْفُ شَرْطٍ | فِعْل وَفَاعِل (ت) |
| أَجْتَهِدُ | إِنْ | اجْتَهَدْتَ |

---

## Summary Table: Conditional Sentences

| Type | شَرْط (Condition) | جَوَابُ الشَّرْطِ (Result) |
|------|-------------------|---------------------------|
| **Zero Conditional** | If + you work hard (Present) | you are successful (Present) |
| **First Conditional** | If + you work hard (Present) | you will be successful (Future) |

The context will help determine the meaning of the Arabic sentence.

## Practice Examples

### With إِنْ
1. إِنْ تَشْكُرُوا اللهَ يَزِدْكُمْ - If you thank Allah, He will increase you
2. أَنَّى تَكُنْ يَأْتِكَ المَوْتُ - Wherever you are, death will come to you
3. إِنْ تُؤْمِنْ بِاللهِ تَدْخُلِ الجَنَّةَ - If you believe in Allah, you will enter Paradise
4. إِنْ تَدْعُ اللهَ يَسْتَجِبْ دُعَاءَكَ - If you call upon Allah, He will answer your supplication
5. وَاللهِ إِنْ تَحْفَظْ لِسَانَكَ تَسْلَمْ - By Allah, if you guard your tongue, you will be safe

### With مَنْ
6. كَيْفَ تُعَامِلُوا النَّاسَ يُعَامِلُوكُمْ - However you treat people, they will treat you
7. إِنْ تَسْتَغْفِرِ اللهَ يَغْفِرْ لَكَ ذُنُوبَكَ - If you seek forgiveness from Allah, He will forgive your sins
8. أَيْنَ تَعْمَلْ يَكْتُبْهُ اللهُ وَيُحَاسِبْكَ بِهِ - Wherever you do (any deed), Allah writes it and will account you for it
9. كَمْ آيَةٍ يَحْفَظْ طَالِبٌ يَسْمَعْهُ المُعَلِّمُ - However many verses a student memorizes, the teacher will hear him
10. وَاللهِ إِنْ تَصْدُقِ النَّاسَ يُبَارِكِ اللهُ فِيكَ - By Allah, if you are truthful with people, Allah will bless you

### With Negation
1. إِنْ لَمْ تَجْتَهِدْ لَا تَفُزْ - If you do not work hard, you will not succeed
2. إِنْ لَمْ تُطِعْنَ أَبَوَيْكُنَّ يَحْزَنَا - If you (fem. pl.) do not obey your parents, they will be sad
3. إِنْ لَمْ تَصْدُقِي لَا يُحِبُّكِ النَّاسُ - If you (fem.) do not speak the truth, people will not love you
4. إِنْ لَمْ تُسْرِعْ يَكُنْ جَوَابُكَ صَحِيحًا لَا يَغْضَبْ عَلَيْكَ أُسْتَاذُكَ بَلْ يَنْصُرْكَ - If you do not hurry (and) your answer is correct, your teacher will not be angry with you, but will help you

### Fronted Result (جَوَاب مُقَدَّم)
1. كُلْ إِنْ كُنْتَ جَائِعًا - Eat if you are hungry
2. لَا تَصُمْ إِنْ كُنْتَ مَرِيضًا - Do not fast if you are sick
3. ﴿اِتَّقُوا اللهَ إِنْ كُنْتُمْ مُؤْمِنِينَ﴾ - Fear Allah if you are believers
4. أُنْصُرِ الفُقَرَاءَ وَالمَسَاكِينَ إِنْ كَانَ عِنْدَكَ مَالٌ - Help the poor and needy if you have wealth

## Quranic Examples

1. ﴿إِنْ كُنْتُمْ تُحِبُّونَ اللهَ فَاتَّبِعُونِي﴾ - If you love Allah, then follow me
2. ﴿وَإِنْ كَذَّبُوكَ فَقُلْ لِي عَمَلِي وَلَكُمْ عَمَلُكُمْ﴾ - And if they deny you, then say: "For me is my work, and for you is your work"
3. ﴿إِنْ تَصْبِرُوا وَتَتَّقُوا فَسَوْفَ يُؤْتِيكُمُ اللهُ أَجْرًا عَظِيمًا﴾ - If you are patient and fear Allah, Allah will give you a great reward
4. ﴿وَإِنْ تَكْفُرُوا فَإِنَّ لِلَّهِ مَا فِي السَّمَاوَاتِ وَالأَرْضِ﴾ - And if you disbelieve, then to Allah belongs whatever is in the heavens and earth
5. ﴿وَإِنْ تَدْعُوهُمْ إِلَى الهُدَى لَا يَسْمَعُوا﴾ - And if you call them to guidance, they do not hear

## Key Terminology

| Arabic Term | English | Description |
|-------------|---------|-------------|
| شَرْط | Condition | The "if" clause |
| جَوَابُ الشَّرْطِ | Result | The "then" clause |
| أَدَاةُ الشَّرْطِ | Conditional conjunction | If, when, whoever, etc. |
| أَسْمَاءُ الشَّرْطِ | Noun conditional conjunctions | مَنْ، مَا، أَيْنَ، etc. |
| حُرُوفُ الشَّرْطِ | Particle conditional conjunctions | إِنْ، لَوْ، لَوْلَا |
| رَابِطَة | Connector | فَ used to connect clauses |
| مَجْزُوم | Jussive case | State of verbs after إِنْ and أَسْمَاءُ الشَّرْطِ |
| جَوَابٌ مُقَدَّم | Fronted result | Result clause before condition |
| شَرْطٌ مُؤَخَّر | Delayed condition | Condition clause after result |
| عَامِل | Governing | Has grammatical effect |
| غَيْرُ عَامِل | Non-governing | No grammatical effect |
